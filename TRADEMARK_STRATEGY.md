# Trademark Strategy

## Important
The WIPO Madrid System is for **international trademark protection**, not for
patents and not for copyright registration.

## Recommended Use
Use Madrid for:
- project name protection
- product name protection
- logo protection
- brand extensions

Possible marks to evaluate:
- RitualMesh
- RitualMesh Decentralized Stock Exchange
- associated logos and word marks

## Note
Trademark filing strategy should be coordinated with the country where the base
application or registration is filed.
