# Patent Claims

## Independent Claim
A system for decentralized transaction clearing using domain-bound cryptographic keys within a sovereign control environment.

## Dependent Claims

1. Each node is associated with a domain identity and key.
2. Transactions are validated before settlement.
3. System state is recorded as canonical truth.
4. Nodes replicate architecture for scaling.
5. Secondary nodes verify integrity.

## Notes
These claims are drafted for provisional filing and future expansion.
