# Legal Setup Notes

## What this bundle does
- installs project legal files
- prepares a key identity template
- generates local signing key material if tools are available

## What this bundle does not do
- it does not file with WIPO
- it does not submit a Madrid application
- it does not create a patent filing
- it does not replace legal advice

## Recommended next steps
1. push the legal files into the repo
2. keep `.secrets/` out of git
3. decide your trademark filing strategy
4. decide whether to pursue patent filing before publishing deeper invention details
