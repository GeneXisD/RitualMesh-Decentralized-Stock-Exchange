# System Preservation Note

The original RitualMesh architecture remains authoritative.

These blueprint documents are an interpretive engineering layer for:
- prototyping
- diagramming
- implementation planning
- licensing and build boundary clarification

They do not replace the original system design. They exist to help convert the current architecture into a reproducible prototype and a diagram-ready specification while preserving the intent, naming, and historical structure already established in the repository.
