# Original Architecture Map

## Canonical Core
- Intel Ubuntu 14.04 federated node
- ClearingHouse / Viacoin execution environment
- Authoritative state and canonical truth

## Secondary Validation Layer
- M4 host
- synchronization
- verification
- continuity / failover preparation
- orchestration support

## Node Cell Architecture
A node cell is defined as:

- Xbox running Dyne:bolic 1.x
- PlayStation running Puredyne
- ASUS AC1900 router

### Node Roles
#### Xbox Bootstrap Node
- legacy bootstrap layer
- initial system preparation
- historical source / artifact staging

#### PlayStation Persistent Build Node
- Puredyne live environment
- persistence and reusable system state
- build carry-forward environment
- candidate nest / live-build role

#### Router Mesh Boundary Node
- transport
- segmentation
- routing boundary
- node-to-node interconnect

## Future / Conditional Layer
- Dyne:bolic 3.x as a source-verified future network integration layer
- Netsukuku integration path
- GNUnet overlay / privacy path

## Support and Governance Layers
- legal and authorship structure
- licensing map
- trademark and policy materials
- build / test / snapshot / sync scripts
