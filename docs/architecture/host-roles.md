# Host Roles

## Intel primary
- primary federated host
- Ubuntu 14.04
- canonical ports
- canonical credentials
- canonical write path

## M4 secondary
- support host
- secondary ClearingHouse-capable environment
- config mirror
- audit and sync
- recovery and migration support

## Android legacy
- legacy portable shell node
- optional helper and control surface
- not canonical financial authority

## Rule
Do not describe all hosts as equal federated identities.
