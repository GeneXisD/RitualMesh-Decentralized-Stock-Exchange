# Vending Machine Role

## Context
Some historical ClearingHouse role descriptions include future or unfinished roles such as a vending machine or block explorer.

## Interpretation
Treat these as planned or conceptual service roles rather than canonical node bring-up requirements.

## Project rule
Do not depend on unfinished future roles to make the primary federated node operational.

## Placement
If needed later, model it as a separate service:
- `services/vending-node/`
- its own docs and API notes
